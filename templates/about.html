PitPixie PitPixie is an AI-powered document analysis tool designed to help users explore large Mine Closure Plan documents more easily. The system allows users to upload documents and ask questions about their content using natural language. PitPixie uses Microsoft Azure AI technologies and a Retrieval-Augmented Generation (RAG) approach to retrieve relevant information from documents and generate helpful responses. It also supports document filtering, chat history, and page referencing to make navigating complex technical reports easier. This prototype was developed to explore how artificial intelligence can improve access to information within large environmental and mining documents.

✨ Features include:

  • Chat with an AI agent about uploaded documents.
  • Upload and analyze large PDF documents.
  • Filter documents by mine site.
  • Export chat results (CSV format).
  • Persistent chat history.
  • Page number referencing for document responses.
  • Secure secret management using Azure Key Vault.
  • Configurable user interface with theme support.
